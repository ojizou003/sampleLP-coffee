# Ojizou Coffee - Premium Coffee Beans LP

厳選された最高品質のコーヒー豆をお届けする「Ojizou Coffee」のランディングページ
です。

## プロジェクト概要

このプロジェクトは、コーヒー豆販売サイトのランディングページ（LP）です。ユーザー
にブランドの魅力、製品ラインナップ、そしてコーヒーへのこだわりを伝えることを目的
としています。

## 特徴

- **レスポンシブデザイン**: PC、タブレット、スマートフォンなど、あらゆるデバイス
  で美しく表示されます。
- **モダンな UI/UX**: シンプルかつ高級感のあるデザインで、ユーザー体験を重視して
  います。
- **セクション構成**:
  - **Hero Section**: ファーストビューでブランドイメージを訴求。
  - **About**: 鮮度、厳選された豆、熟練の技術など、選ばれる理由を紹介。
  - **Products**: おすすめのコーヒー豆（Signature Blend, Espresso Roast, Decaf
    Selection）を紹介。
  - **Process**: 豆の選定から焙煎、抽出までのプロセスをタイムライン形式で紹介。
  - **Contact**: お問い合わせフォーム。

## 技術スタック

- **HTML5**: セマンティックなマークアップ。
- **CSS3**: カスタムプロパティ（変数）、Flexbox、Grid Layout を使用したスタイリ
  ング。
- **JavaScript (Vanilla)**: ハンバーガーメニューの開閉やスクロールアニメーション
  などのインタラクション。
- **Google Fonts**: Noto Sans JP を使用。
- **Unsplash**: 高品質なイメージ画像。

## セットアップと使用方法

このプロジェクトは静的な HTML/CSS/JS サイトです。特別なビルドプロセスは必要あり
ません。

1.  リポジトリをクローンまたはダウンロードします。
2.  `index.html` をブラウザで開きます。

## ディレクトリ構成

```
.
├── index.html  # メインのHTMLファイル
├── style.css   # スタイルシート
├── script.js   # JavaScriptファイル
└── README.md   # プロジェクト説明書（このファイル）
```
